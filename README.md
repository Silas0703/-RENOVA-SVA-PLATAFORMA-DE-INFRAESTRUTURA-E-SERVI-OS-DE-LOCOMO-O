Descrição do Site Renova SVA
♻️ Renova SVA — Plataforma de Infraestrutura e Serviços de Locomoção Sustentável
O que é o Renova SVA?

O Renova SVA é uma plataforma web inovadora desenvolvida para promover a mobilidade urbana sustentável. O sistema permite que usuários comuns aluguem veículos elétricos de forma simples, rápida e intuitiva, contribuyendo para a redução da emissão de poluentes e para um meio ambiente mais limpo.
Para que serve?

A plataforma foi criada com o objetivo de facilitar o acesso a veículos elétricos sustentáveis, conectando usuários que precisam de uma opção de transporte ecológicos às empresas ou administradores que oferecem a frota de veículos. Além disso, o sistema conta com um painel administrativo completo para gestãoda frota, controle de locações e monitoramento em tempo real.
Quais são as principais funcionalidades?

1. Catálogo de Veículos A plataforma possui um catálogo completo com diferentes tipos de veículos elétricos, incluindo scooters, bikes, monopatinetes e carros. Cada veículo possui informações detalhadas sobre autonomia, modelo, categoria e preço por hora ou por quilômetro.

2. Sistema de Locação O usuário pode selecionar um veículo disponível, realizar a locação com apenas alguns cliques e receber a confirmação na hora. O sistema controla automaticamente o tempo de uso e calcula o valor total no momento da devolução.

3. Estações de Recarga A plataforma informa a localização das estações de recarga próximas, com endereço, tipo de carregador disponível e número de vagas livres.

4. Dashboard Administrativo Um painel completo que exibe estatísticas em tempo real, como total de veículos, quantidade disponível, veículos em uso, locações ativas e número de estações cadastradas.

5. Feedback ao Usuário Mensagens claras e objetivas que informam o usuário sobre o andamento de suas solicitações, facilitando a experiência de uso.
Quem pode usar?

    Usuários comuns: Pessoas que precisam de uma opção de transporte sustentável e econômica.
    Administradores: Gestores de frota que desejam controlar veículos, locações e estações.
    Empresas de mobilidade: Organizações que oferecem serviços de locação de veículos elétricos.

Tecnologias utilizadas

O projeto foi desenvolvido utilizando tecnologias modernas e amplamente utilizadas no mercado de trabalho:

    Frontend: React.js — Uma das bibliotecas mais populares para desenvolvimento de interfaces web.
    Backend: Python com FastAPI — Framework rápido e moderno para criação de APIs.
    Estilização: CSS3 com design moderno e cores que remetem à sustentabilidade.
    Comunicação: Axios — Biblioteca para requisições HTTP.

Como acessar?

O projeto está configurado para rodar localmente em duas partes:

    Frontend: Executado no endereço http://localhost:3000
    Backend: Executado no endereço http://localhost:8000

A documentação completa da API está disponível em http://localhost:8000/docs.
Objetivos acadêmicos

Este projeto foi desenvolvido como parte de um trabalho de faculdade com os seguintes objetivos pedagógicos:

    Praticar o desenvolvimento web com tecnologías atuais
    Criar uma API RESTful completa e funcional
    Implementar uma interface responsiva e intuitiva
    Demonstrar a arquitetura Cliente-Servidor
    Preparar o aluno para projetos reais de desenvolvimento
